---
title: "素材生视频"
api: "POST /api/v1/task/submit"
description: "Jimeng / Seedance 素材生视频 — 基于图片、视频、音频等素材生成视频"
---

通过传入图片、视频、音频等素材生成视频。根据素材数量和类型自动选择生成模式。

### 支持的素材类型

| 类型 | 支持格式 | 说明 | 支持模型 |
|------|---------|------|---------|
| 图片 | `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.bmp` | 用于首尾帧或参考图 | 所有模型 |
| 视频 | `.mp4`, `.mov`, `.m4v` | 用于多模态参考 | 仅 `seedance-*` |
| 音频 | `.mp3`, `.wav` | 用于多模态参考 | 仅 `seedance-*` |

<Warning>
  **视频和音频素材仅 Seedance 模型支持**（`seedance-2.0`、`seedance-2.0`、`seedance-2.0-fast`）。`jimeng-video-*` 模型仅支持最多 2 张图片（首尾帧模式）。
</Warning>

<ParamField body="model" type="string" required>
  模型名称，如 `seedance-2.0`、`seedance-2.0-fast`、`jimeng-video-3.5-pro`。
</ParamField>

<ParamField body="callBackUrl" type="string">
  Webhook 回调地址。
</ParamField>

<ParamField body="input" type="object" required>
  <Expandable title="properties" defaultOpen>
    <ParamField body="prompt" type="string" required>
      提示词。可用 `@1`、`@2`、`@3` 引用对应位置的素材（也支持 `@图1`、`@image1` 写法）。
    </ParamField>
    <ParamField body="mediaUrls" type="string[]">
      素材 URL 列表。支持图片、视频、音频混合传入，素材类型根据文件扩展名自动检测。素材的数量和类型决定生成模式（见下方说明）。
    </ParamField>
    <ParamField body="imageUrls" type="string[]">
      参考图片 URL 列表。向后兼容，等同于仅含图片的 `mediaUrls`。

      > 如果同时提供 `mediaUrls` 和 `imageUrls`，以 `mediaUrls` 为准。
    </ParamField>
    <ParamField body="ratio" type="string">
      视频比例。可选值：`1:1`、`4:3`、`3:4`、`16:9`、`9:16`。
    </ParamField>
    <ParamField body="resolution" type="string">
      视频分辨率。可选值：`480p`、`720p`、`1080p`。
    </ParamField>
    <ParamField body="duration" type="integer">
      视频时长（秒）。Seedance 模型支持 `4`–`15` 秒；其他模型支持 `5`、`10`、`15`。
    </ParamField>
    <ParamField body="fallback" type="boolean" default="true">
      Seedance 模型失败后是否自动降级到 `jimeng-video-3.5-pro`。默认 `true`。设为 `false` 则禁止降级，直接返回原始错误。仅对 Seedance 系列模型生效，其他模型忽略此字段。注意：多模态请求（包含视频/音频素材、超过 2 张图片、或提示词含 `@1`/`@2` 等占位符引用）无论此值如何都不会降级。
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```bash 首帧模式（1 张图片）
curl --request POST \
  --url https://api.maxapi.io/api/v1/task/submit \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{

  "input": {
    "prompt": "吹灭蜡烛",
    "mediaUrls": ["https://example.com/photo.jpg"]
  }
}'
```

```bash 多模态（图片 + 音频）
curl --request POST \
  --url https://api.maxapi.io/api/v1/task/submit \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \

  "model": "seedance-2.0",
  "input": {
    "prompt": "@1 随着 @2 的节奏跳舞",
    "mediaUrls": [
      "https://example.com/dancer.jpg",
      "https://example.com/music.mp3"
    ]
  }

```

## 生成模式

根据素材的数量和类型，自动选择不同模式：

| 条件 | 模式 | 说明 | 支持模型 |
|------|------|------|---------|
| 1 张图片 | 首帧模式 | 图片作为视频首帧 | 所有模型 |
| 2 张图片 | 首尾帧模式 | 两张图片分别作为首帧和尾帧 | 所有模型 |
| 3+ 张图片 | 多图参考模式 | 通过 `@1`、`@2`、`@3` 引用多张图片 | 仅 `seedance-*` |
| 包含视频或音频 | 多模态参考模式 | 图片、视频、音频混合作为参考素材 | 仅 `seedance-*` |

### 1 张图 — 首帧模式

以图片作为视频的第一帧，根据 prompt 生成后续动画。

```json
{
  "model": "seedance-2.0",
  "input": {
    "prompt": "吹灭蜡烛",
    "mediaUrls": ["https://assets.movart.ai/landingpage/dog.webp"]
  }
}
```

### 2 张图 — 首尾帧模式

第一张图作为视频首帧，第二张图作为视频尾帧，AI 自动生成中间过渡动画。

```json
{
  "model": "seedance-2.0",
  "input": {
    "prompt": "变身",
    "mediaUrls": [
      "https://assets.movart.ai/3D-Cartoon.jpg",
      "https://assets.movart.ai/landingpage/dog.webp"
    ]
  }
}
```

### 3+ 张图 — 多图参考模式（仅 Seedance）

prompt 中用 `@1`、`@2`、`@3` 引用对应位置的素材。

```json
{
  "model": "seedance-2.0",
  "input": {
    "prompt": "@1 和 @2 在聊天，@3 在旁边看",
    "mediaUrls": [
      "https://example.com/person-a.jpg",
      "https://example.com/person-b.jpg",
      "https://example.com/person-c.jpg"
    ]
  }
}
```

### 多模态参考模式（仅 Seedance）

当素材中包含视频或音频（或图片 + 视频 + 音频的任意混合）时，自动进入多模态参考模式。用 `@1`、`@2` 等引用对应位置的素材。

**图片 + 音频** — 让角色随音乐节奏舞动：

```json
{
  "model": "seedance-2.0",
  "input": {
    "prompt": "@1 随着 @2 的节奏跳舞",
    "mediaUrls": [
      "https://example.com/dancer.jpg",
      "https://example.com/music.mp3"
    ],
    "ratio": "16:9",
    "duration": 8
  }
}
```

**图片 + 视频** — 以视频作为动作参考，结合图片角色：

```json
{
  "model": "seedance-2.0",
  "input": {
    "prompt": "@1 做出 @2 中的动作",
    "mediaUrls": [
      "https://example.com/character.jpg",
      "https://example.com/reference-motion.mp4"
    ]
  }
}
```

**多类型混合** — 自由组合图片、视频和音频：

```json
{
  "model": "seedance-2.0",
  "input": {
    "prompt": "@1 和 @2 随着 @3 一起跳舞",
    "mediaUrls": [
      "https://example.com/person-a.jpg",
      "https://example.com/person-b.jpg",
      "https://example.com/bgm.mp3"
    ],
    "duration": 10
  }
}
```
