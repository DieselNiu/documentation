通过传入 `mediaUrls`（或纯图片场景使用 `imageUrls`），根据素材数量和类型自动选择不同的生成模式。适用于所有 `seedance-*` 和 `jimeng-video-*` 模型。

<ParamField body="model" type="string" required>
  模型名称，如 `seedance-2.0-pro`、`seedance-2.0-fast`、`jimeng-video-3.5-pro`。
</ParamField>

<ParamField body="callBackUrl" type="string">
  Webhook 回调地址。
</ParamField>

<ParamField body="input" type="object" required>
  <Expandable title="properties" defaultOpen>
    <ParamField body="prompt" type="string" required>
      提示词。多素材模式下可用 `@1`、`@2`、`@3` 引用对应位置的素材。
    </ParamField>
    <ParamField body="mediaUrls" type="string[]">
      素材 URL 列表，支持图片/视频/音频混合。素材类型通过 URL 扩展名自动检测。数量和类型决定生成模式（见下方说明）。
    </ParamField>
    <ParamField body="imageUrls" type="string[]">
      参考图片 URL 列表（向后兼容，等同于仅含图片的 `mediaUrls`）。如果同时提供 `mediaUrls` 和 `imageUrls`，以 `mediaUrls` 为准。
    </ParamField>
    <ParamField body="ratio" type="string">
      视频比例。可选值：`1:1`、`4:3`、`3:4`、`16:9`、`9:16`。
    </ParamField>
    <ParamField body="resolution" type="string">
      视频分辨率。可选值：`480p`、`720p`、`1080p`。
    </ParamField>
    <ParamField body="duration" type="integer">
      视频时长（秒）。可选值：`5`、`10`、`15`。
    </ParamField>
  </Expandable>
</ParamField>

<Note>
  `mediaUrls` 和 `imageUrls` 二选一。新开发推荐使用 `mediaUrls`；如果两者都提供，以 `mediaUrls` 为准。
</Note>

### 支持的素材类型

| 类型 | 支持格式 | 说明 |
|------|---------|------|
| 图片 | `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.bmp` | 用于首尾帧或参考图 |
| 视频 | `.mp4`, `.mov`, `.m4v` | 用于多模态参考 |
| 音频 | `.mp3`, `.wav` | 用于多模态参考 |

<RequestExample>
```bash 首帧模式（1 张图片）
curl --request POST \
  --url https://api.maxapi.io/api/v1/task/submit \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
  "model": "seedance-2.0-pro",
  "input": {
    "prompt": "吹灭蜡烛",
    "mediaUrls": ["https://example.com/photo.jpg"]
  }
}'
```

```bash 多模态（图片 + 音频）
curl --request POST \
  --url https://api.maxapi.io/api/v1/task/submit \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
  "model": "seedance-2.0",
  "input": {
    "prompt": "使用 @1 角色和 @2 背景音乐生成视频",
    "mediaUrls": [
      "https://example.com/character.jpg",
      "https://example.com/bgm.mp3"
    ]
  }
}'
```
</RequestExample>

<ResponseExample>
```json 200 提交成功
{
  "code": 0,
  "msg": "ok",
  "data": {
    "taskId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
  }
```

</ResponseExample>

## 生成模式

根据素材数量和类型，自动选择不同模式：

| 条件 | 模式 | 说明 |
|------|------|------|
| 0 个素材 | 文生视频 | 纯文本生成 |
| 1-2 个纯图片 | 首尾帧 | 第 1 张为首帧，第 2 张为尾帧 |
| 3+ 个素材，或包含视频/音频 | 多模态参考 | 图片/视频/音频混合参考 |

### 1 张图 — 首帧模式

以图片作为视频的第一帧，根据 prompt 生成后续动画。

```json
{
  "model": "seedance-2.0-pro",
  "input": {
    "prompt": "吹灭蜡烛",
    "mediaUrls": ["https://assets.movart.ai/landingpage/dog.webp"]
  }
}
```

### 2 张图 — 首尾帧模式

第一张图作为视频首帧，第二张图作为视频尾帧，AI 自动生成中间过渡动画。

```json
{
  "model": "seedance-2.0-pro",
  "input": {
    "prompt": "变身",
    "mediaUrls": [
      "https://assets.movart.ai/3D-Cartoon.jpg",
      "https://assets.movart.ai/landingpage/dog.webp"
    ]
  }
}
```

### 3+ 张图 — 多图参考模式

prompt 中用 `@1`、`@2`、`@3` 引用对应位置的素材。

```json
{
  "model": "seedance-2.0",
  "input": {
    "prompt": "@1 和 @2 在聊天，@3 在旁边看",
    "mediaUrls": [
      "https://assets.movart.ai/3D-Cartoon.jpg",
      "https://assets.movart.ai/landingpage/dog.webp",
      "https://assets.movart.ai/landingpage/dog.webp"
    ]
  }
}
```

### 多模态参考（图片 + 视频 / 音频混合）

Seedance 2.0 支持图片、视频、音频混合上传，素材类型通过 URL 扩展名自动检测。

```json
{
  "model": "seedance-2.0",
  "input": {
    "prompt": "使用 @1 角色和 @2 背景音乐生成视频",
    "mediaUrls": [
      "https://example.com/character.jpg",
      "https://example.com/bgm.mp3"
    ]
  }
}
```

<Note>
  以上模式适用于所有 `seedance-*` 和 `jimeng-video-*` 系列模型。`imageUrls` 仍可用于纯图片场景（向后兼容），新开发推荐使用 `mediaUrls`。
</Note>
