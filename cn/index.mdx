## 核心特性

<Columns cols={3}>
  <Card title="统一接口" icon="plug">
    所有模型共享相同的提交 / 查询 / 取消接口，切换模型只需更改 model 参数。
  </Card>
  <Card title="异步任务" icon="clock">
    提交后立即返回 taskId，通过轮询或 Webhook 获取结果。
  </Card>
  <Card title="Webhook 回调" icon="webhook">
    任务完成后自动推送结果到你的服务器，无需轮询。
  </Card>
</Columns>
