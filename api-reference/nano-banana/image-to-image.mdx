---
title: "Image to Image"
api: "POST /api/v1/task/submit"
description: "Nano Banana Image-to-Image — Transform existing images with AI"
---

Use Nano Banana to transform existing images based on text prompts. Apply style transfer, modifications, or creative edits. Available through Flow API or Google Direct channel.

### Available Models

`nano-banana` · `nano-banana-pro` · `nano-banana-google` · `nano-banana-pro-google`

### Channels

<AccordionGroup>
  <Accordion title="Flow Channel — nano-banana / nano-banana-pro">
    Proxied through Flow API with automatic fallback. If Flow fails, the system automatically switches to Google Vertex AI.
  </Accordion>
  <Accordion title="Google Direct — nano-banana-google / nano-banana-pro-google">
    Direct connection to Google Vertex AI official API, bypassing the Flow proxy. The `input` parameters are identical — just change the `model` value to switch channels.
  </Accordion>
</AccordionGroup>

<ParamField body="model" type="string" required>
  Model name, e.g. `nano-banana`, `nano-banana-pro`, `nano-banana-google`, `nano-banana-pro-google`.
</ParamField>

<ParamField body="callBackUrl" type="string">
  Webhook callback URL.
</ParamField>

<ParamField body="input" type="object" required>
  <Expandable title="properties" defaultOpen>
    <ParamField body="prompt" type="string" required>
      Prompt describing the transformation you want to apply.
    </ParamField>
    <ParamField body="ratio" type="string">
      Output image aspect ratio. Options: `16:9`, `9:16`, `1:1`, `4:3`, `3:4`. Default: `16:9`.
    </ParamField>
    <ParamField body="resolution" type="string">
      Output resolution. Options: `1k`, `2k`, `4k`. Default: `1k` (only `nano-banana-pro` supports 2K/4K).
    </ParamField>
    <ParamField body="imageUrls" type="array" required>
      Array containing **exactly 1 input image URL**. Must be publicly accessible HTTPS URL.
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```bash Style Transfer
curl --request POST \
  --url https://api.maxapi.io/api/v1/task/submit \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
  "model": "nano-banana-pro",
  "input": {
    "prompt": "Transform this image into an oil painting style",
    "ratio": "16:9",
    "resolution": "2k",
    "imageUrls": ["https://example.com/input.jpg"]
  }
}'
```

```bash Image Modification
curl --request POST \
  --url https://api.maxapi.io/api/v1/task/submit \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
  "model": "nano-banana-pro",
  "input": {
    "prompt": "Change the background to a sunset beach scene",
    "ratio": "1:1",
    "imageUrls": ["https://example.com/portrait.jpg"]
  }
}'
```
</RequestExample>

<Warning>
  **Input Image Requirements:**
  - Formats: JPEG, PNG, WebP
  - Must be publicly accessible HTTPS URL
  - Recommended size: < 10MB
  - The output aspect ratio and resolution are independent of the input image
</Warning>
